# Chèn ảnh từ bên ngoài vào

Các vấn đề bao gồm:

- Định dạng tệp
- Tạo folder lưu trữ ảnh
- Kích thước ảnh
- Chèn ảnh (từ một ảnh cho đến nhiều ảnh)
- Căn chỉnh ảnh
- Khi nào thì nên sử dụng lệnh này vào tài liệu nào ?

## Gói và chèn ảnh bên ngoài cơ bản

```latex
\usepackage{graphicx} % Gói
```

. . . <!--Viêt sau-->

```latex
\includegraphics{options}{filename}
```

Các tham số <!--Xem lại thuật ngữ sử dụng--> thường dùng của tùy chọn (`options`):

| `options`          | Chức năng                                                              |
| :----------------- | :--------------------------------------------------------------------- |
| width = len        | phóng hoặc thu về chiều rộng _len_                                     |
| height = len       | phóng hoặc thu chiều cao _len_                                         |
| scale = x          | phóng hoặc thu                                                         |
| angle = x          | quay ảnh một góc $x^o$                                                 |
| trim = lx by rx ty | cắt ảnh góc dưới bên trái là (lx, by) và góc trên bên phải là (rx, ty) |
| draf               | không hiển thị ảnh, chỉ hiển thi khung xung quang                      |

## Môi trường hình ảnh <!--Đặt tiêu đề lại sau-->

. . .<!--Viêt sau-->

```latex
\begin{figure}[] % tham số
    \centering % Căn ảnh ở giữa
    \includegraphics{}{} % chèn ảnh
    \caption{}
    \label{} % Nhãn và tham chiếu
\end{figure}
```

trong đó:

| :------ | :-------- |
| h       |           |
| t       |           |
| b       |           |
| p       |           |
| !       |           |
| H       |           |

## Chèn nhiều hình ảnh trong một hình <!--Đặt tiêu đề lại sau-->

Có thể chèn nhiều hình ảnh vào cùng một hình <!--viết lại sau-->, với mỗi hình ảnh có tham chiếu và nhãn riêng. <!--Viết lại sau-->

. . . <!--Viêt sau-->

```latex
\usepackage{subcaption}
```

. . . <!--Viêt sau-->

```latex
\begin{figure}[]
    \begin{subfigure}{. . .\textwidth} % Bắt buộc phải căn chiều rộng của hình khi sử dụng
    \includegraphics{}{} % chèn ảnh
    \caption{}
    \label{}
    \end{subfigure}
    \begin{subfigure}{. . .\textwidth} % Bắt buộc phải căn chiều rộng của hình khi sử dụng
    \includegraphics{}{} % chèn ảnh
    \caption{}
    \label{}
    \end{subfigure}

    \caption{}
    \label{}
\end{figure}
```
